# date-analysis
作业# 商品销售数据分析项目

## 一、项目背景
随着电商行业快速发展，行业竞争日益激烈。本项目利用Python对某运动用品店铺的销售数据进行分析，为店铺运营决策提供数据支持。

## 二、数据说明
- 数据文件：商品销售数据.xlsx
- 包含两个工作表：信息表（39条商品记录）、销售数据表（2040条订单记录）
- 主要字段：商品大类、商品小类、商品编号、商品销售价、订单日期、订单数量等

## 三、分析方法
1. 使用pandas读取并合并两个数据表
2. 计算销售金额，提取月份信息
3. 按月份、商品大类、商品小类进行分组统计
4. 使用matplotlib绘制折线图、柱状图进行可视化

## 四、分析结论
1. **月度趋势**：全年销售整体上升，7月和10-11月为销售高峰，1月为低谷
2. **品类对比**：运动上装占比62%，是绝对主力；运动下装占28%；运动配饰占10%
3. **爆款商品**：外套、背心、短袖销售额排名前三
4. **经营建议**：高峰期提前备货，重点推广主力品类，淡季制定促销策略
